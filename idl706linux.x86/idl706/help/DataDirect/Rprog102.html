Driver Requirements

To access a Progress OpenEdge Release 10 database with the OpenEdge driver, your client must have the Progress OpenEdge 10 client or higher installed.

NOTE: The Progress OpenEdge driver does not support Progress databases created prior to OpenEdge Release 10.

Before using the driver, you must set the DLC environment variable to your OpenEdge DLC directory. For example: 

set DLC=C:\OpenEdge
