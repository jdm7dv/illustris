The Oracle Wire Protocol Driver

The DataDirect Connect for ODBC Oracle Wire Protocol driver (the Oracle Wire Protocol driver) supports the following Oracle database servers:

  • Oracle 8i R2, R3 (8.1.6, 8.1.7)
  • Oracle 9i R1 and R2 (9.0.1 and 9.2)
  • Oracle 10g R1 (10.1)

The Oracle Wire Protocol driver is supported in the Windows and UNIX environments. See "Environment-Specific Information" for detailed information about the Windows and UNIX environments supported by this driver.

See the README file shipped with your DataDirect product for the file name of the Oracle Wire Protocol driver.

NOTE: The Oracle Wire Protocol driver does not require any Oracle client software. DataDirect also provides an Oracle client-based driver that can access earlier versions of Oracle databases. See "The Oracle Driver" for details.
