ODBC Conformance Level

See "ODBC API and Scalar Functions" for a list of the API functions supported by the Oracle driver.

The Oracle driver also supports the following functions:

  • SQLColumnPrivileges
  • SQLDescribeParam (if EnableDescribeParam=1)
  • SQLForeignKeys
  • SQLPrimaryKeys
  • SQLProcedures
  • SQLProcedureColumns
  • SQLSetPos
  • SQLTablePrivileges

The driver supports the core SQL grammar.
