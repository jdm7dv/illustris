About DataDirect Connect® for ODBC Drivers

The DataDirect Connect for ODBC drivers are compliant with the Open Database Connectivity (ODBC) specification.

Support for Multiple Environments

DataDirect Technologies provides ODBC-compliant database drivers for Windows and UNIX operating systems. Refer to "Environment-Specific Information" for an explanation of the environment-specific differences when using the database drivers in your operating environment.

Database drivers are continually being added to each operating environment. For the latest information about the specific drivers available for your platform, see the README file in your software package, or refer to the DataDirect Technologies World Wide Web page at:
