ODBC Conformance Level

See "ODBC API and Scalar Functions" for a list of the API functions supported by the Sybase Wire Protocol driver. In addition, the following functions are supported:

  • SQLColumnPrivileges
  • SQLForeignKeys
  • SQLPrimaryKeys
  • SQLProcedureColumns
  • SQLProcedures
  • SQLTablePrivileges

The driver supports the minimum SQL grammar.
