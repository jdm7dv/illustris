The Sybase Wire Protocol Driver

The DataDirect Connect for ODBC Sybase Wire Protocol driver (the Sybase Wire Protocol driver) supports the following Sybase database servers:

  • Sybase Adaptive Server 11.5 and higher
  • Sybase Adaptive Server Enterprise 12.0 and 12.5 and higher

The Sybase Wire Protocol driver is supported in the Windows and UNIX environments. See "Environment-Specific Information" for detailed information about the Windows and UNIX environments supported by this driver.

See the README file shipped with your DataDirect product for the file name of the Sybase Wire Protocol driver.
