The Btrieve (Pervasive.SQL) Driver

The DataDirect Connect for ODBC Btrieve driver (the Btrieve driver) supports the following versions of Btrieve files:

  • Btrieve version 6.15
  • Pervasive.SQL 7.0
  • Pervasive.SQL 2000

The driver executes SQL statements directly on Btrieve databases.

The Btrieve driver is supported in the Windows environments. See "Environment-Specific Information" for detailed information about the Windows environments supported by this driver.

See the README file shipped with your DataDirect product for the file name of the Btrieve driver.
