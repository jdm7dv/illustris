The SQL Server Wire Protocol Driver

The DataDirect Connect for ODBC SQL Server Wire Protocol driver (the SQL Server Wire Protocol driver) supports:

Microsoft SQL Server 7.0 and SQL Server 2000 with Service Packs 1, 2, and 3.

The SQL Server Wire Protocol driver is supported in the Windows and UNIX environments. See "Environment-Specific Information" for detailed information about the Windows and UNIX environments supported by this driver.

See the README file shipped with your DataDirect product for the file name of the SQL Server Wire Protocol driver.
