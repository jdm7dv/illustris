Driver Requirements

There are no client requirements for the SQL Server Wire Protocol driver.

Windows

For support of version 7.0 and 2000 of SQL Server, the driver requires the SQL Server 7.0 versions of Net-Library DLL files, which are installed when you install the SQL Server Wire Protocol driver. The driver communicates with network software through the SQL Server Net-Library interface.

UNIX

To use the SQL Server Wire Protocol driver on UNIX, you must have TCP/IP configured on both the UNIX client and the Windows server on which the SQL Server database resides. The UNIX SQL Server TCP/IP network client library is built into the SQL Server Wire Protocol driver on UNIX.

The SQL Server Client configuration has been merged with the ODBC driver configuration and is set in the system information file.
