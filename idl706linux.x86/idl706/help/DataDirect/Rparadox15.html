ODBC Conformance Level

See "ODBC API and Scalar Functions" for a list of the API functions supported by the Paradox driver. In addition, the following functions are supported:

  • SQLSetPos
  • SQLPrimaryKeys

When used with Paradox 5 or Paradox 7, 8, 9, and 10 database files, the Paradox driver supports the SQLForeignKeys function. The Paradox driver also supports backward and random fetching in SQLExtendedFetch and SQLFetchScroll. The Paradox driver supports the minimum SQL grammar.
