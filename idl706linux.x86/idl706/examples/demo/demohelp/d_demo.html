IDL Demo Library

This program serves as a demonstration of the capabilities and features of IDL, the Interactive Data Language. IDL is data analysis, visualization, and application development software developed by ITT Visual Information Solutions. This demonstration was developed using IDL to illustrate the unparalleled flexibility offered for reading, analyzing, and displaying any type of data.

The IDL source code for the demo programs can be found in the IDL distribution under examples/demo/demosrc.
