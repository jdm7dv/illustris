U. S. Census

The "Animate Census" button will cycle through the U.S. Census data from 1790 to 1990. The "Select Census Year" pull-down menu lets you select a specific year of data to be shown. `The "Rotation Angle" slider widget can be used to explicitly define an angle about the Y-axis at which to view a selected. year.

Dwelling the mouse pointer over a State will cause the State to highlight in yellow, and will cause the State's name and population to be displayed. This demonstrates Object Graphics' 3D object selection capability.

The individual State outlines were derived from IDL's map database.

This procedure was authored by the Professional Services Group of ITT Visual Information Solutions. For those desiring a larger implementation, the Professional Services Group can be contacted (psg@ittvis.com) for consulting and custom coding.

About the Data

This demonstration illustrates United States Census Bureau data collected in the decades 1790-1990.

State heights represent total population, while colors indicate relative population change from the preceding decade's census.

States for which no data were collected in the previous decade are displayed in white.
