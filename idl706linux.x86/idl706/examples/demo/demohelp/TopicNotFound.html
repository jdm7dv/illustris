Why Didn't the Help Viewer Find My Topic?

The fact that the IDL Online Help system did not find an exact match for your search term does not mean that no information on your topic is available. When you specify a help topic, IDL first checks to see if the string you entered exactly matches the name of a top-level help topic. (Well, not quite exactly; case is ignored and some characters that might occur in the topic title are replaced.) If a match is found, the topic is displayed. If no exact match is found, IDL searches the index of the online help system for an single entry that exactly matches your search term. If match is found, the topic is displayed. If no match is found during either of these searches, or if your search term corresponds to more than one topic in the index, you see this message.
