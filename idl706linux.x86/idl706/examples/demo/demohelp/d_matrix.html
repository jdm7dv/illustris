Matrices

The Matrix demo visualizes a 3D data set in several different ways:

     

  • Velocity field - Each arrow represents a vector. This type of plot can be used to show ice displacement, slope gradient, oceanic current and more.

    •  

  • Contour plot - The lines connect data items with equal values.

    •  

  • Image - The X and Y coordinates correspond to the value indicated by the axes. The Z coordinate is represented by the color.

    •  

  • Surface plot - The data values are shown in perspective, giving a three dimensional view of the data set.

    •  

  • Show3 - This combines 3 different representations of the same data set. The top has a contour plot. The bottom has an image plot. In the middle, there is a surface mesh.

The plot is drawn with the simple command: 

SHOW3, data
