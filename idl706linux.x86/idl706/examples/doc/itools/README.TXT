This directory contains iTools code used or referred to in
the iTool Developer's Guide. The following types of files
are found in this directory:

iTool Example Files
-------------------

The example files in this directory contain working code
samples for iTools and iTool components. In many cases, the
files in this directory reflect examples created and discussed
in the iTool Developer's Guide.


Utility Files
-------------

itpropertyreport.pro
  Displays the names, data types, and current values of itool
  object properties.


Supporting Files
----------------

This directory also contains several files, such as icon
bitmap files, that are referred to in the example code.
