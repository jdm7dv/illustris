README file: <IDL_Directory>
               license
------------------------------------------------------------

This directory is the default location for the license.dat file used for
Client/Server licensing for Unix and Windows systems.  

For more information about installing and using Client/Server licensing,
consult the Installation and Licensing guide, which is available in
the IDL online help system and in the instlic.pdf fle in the
info/docs directory on your CD-ROM.
